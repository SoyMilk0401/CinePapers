# CinePapers

**CinePapers**는 국내 3대 영화관(CGV, 롯데시네마, 메가박스)의 진행 중인 이벤트 정보를 통합하여 조회하고, 영화 팬들을 위한 기능을 제공하는 데스크톱 애플리케이션입니다.

## 주요 기능

1.  **멀티플렉스 3사 통합 지원**

      * **CGV**, **롯데시네마**, **메가박스**의 이벤트를 하나의 인터페이스에서 전환하며 볼 수 있습니다.
      * 각 영화관별 고유 카테고리(영화, 시사회/무대인사, 제휴/할인, HOT 등) 탭을 지원합니다.

2.  **이벤트 조회 및 검색**

      * 무한 스크롤(Infinite Scroll) 방식을 통해 이벤트 목록을 끊김 없이 로딩합니다.
      * 키워드 검색을 통해 원하는 이벤트를 빠르게 찾을 수 있습니다.

3.  **상세 정보 및 이미지 뷰어**

      * 이벤트 클릭 시 상세 팝업을 통해 고화질 이벤트 이미지를 확인할 수 있습니다.
      * WebView나 브라우저를 띄우지 않고 이미지를 직접 파싱 하여 보여줍니다.
